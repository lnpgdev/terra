About Component Implementation

This demo showcases two implementation patterns for each component:
HTML API: Write raw HTML markup directly. Ideal for static content or server-rendered templates.
Factory API: Use the exported TypeScript utility functions (e.g., createButton()) to generate components dynamically. Recommended for JavaScript-heavy applications.
