Scrollspy

Nav pills tracking a scrollable region
  • Introduction
  • Features
  • Usage
  • Summary
Introduction

Scrollspy observes the scroll position and updates the active nav link to match the visible section. It uses Bootstrap's ScrollSpy plugin under the hood.

Features

Supports configurable root margin and programmatic initialisation via initScrollspy(). Works with any scrollable container.

Usage

Add data-bs-spy="scroll" and data-bs-target to the scrollable container. Bootstrap auto-initialises. Use initScrollspy() for programmatic setup.

Summary

A lightweight wrapper — no additional CSS overrides needed. Bootstrap handles active-link highlighting via the .active class on the matching nav link.
